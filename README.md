# Terminal Coffee Alam Barajo

Website resmi **Terminal Coffee Alam Barajo** — cafe kopi murah berkualitas di kawasan Terminal Alam Barajo, Simpang Rimbo, Kota Jambi. Static website yang dioptimalkan untuk local SEO, mencakup halaman menu, galeri, artikel, dan informasi lokasi.

## Halaman

- `index.html` — Halaman utama
- `artikel.html` — Artikel & konten SEO

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome 6.5
- Google Fonts (Playfair Display, DM Sans, Lora)

## SEO

- JSON-LD Schema (CafeOrCoffeeShop)
- Open Graph & Twitter Card
- Geo meta tags
- Canonical URL
- sitemap.xml & robots.txt
